# 📄 파일 이름 규칙 (필수 ⭐)

## 형식
```
풀이사이트_문제번호(번호 없으면 이름).java
```

## 예시
```
SWEA_1961.java
BOJ_17658.java
```

## 규칙
- 번호 + 문제이름 필수
- 공백 대신 `_` 사용
- test, solution1 같은 이름 금지
